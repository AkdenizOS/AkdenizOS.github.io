Why it matters

A modern academic ecosystem, not just a link list.

AkdenizOS is designed to help students find the right material faster, understand the department roadmap visually, and contribute back without friction.

Organized by real academic progression

Repositories are grouped around the actual student journey — from first-year fundamentals to advanced senior electives and graduation projects.

Built for speed

Students can quickly jump from course code to repository, cutting down the chaos of searching through scattered messages and drives.

Contribution-first mindset

Anyone can enrich the ecosystem by opening an issue, submitting a pull request, or helping launch a missing course repository.

More than lecture notes

The organization also surfaces simulations, systems, visualizers, management apps, and interdisciplinary student builds.

Student knowledge, publicly accessible

Everything is positioned as an open platform so current and future students can build on shared academic momentum.

Academic roadmap

Follow the four-year learning journey.

Year 1

Foundations

Programming, physics, mathematics, statistics, and the first building blocks of engineering thinking.

  • CSE-101T / CSE-101L
  • CSE-102T / CSE-102L
  • CSE-105, CSE-111, CSE-112
  • MAT-163, MAT-164
Year 2

Core engineering

Students move into data structures, OOP, databases, computer organization, discrete math, and digital design.

  • CSE-201 Data Structures
  • CSE-203 OOAD
  • CSE-204 Database Systems
  • CSE-206, CSE-211, CSE-221
Year 3

Systems and specialization

Networking, software engineering, scripting, web development, AI, embedded systems, and interface design expand the toolbox.

  • CSE-320 Computer Networks
  • CSE-332 Software Engineering
  • CSE-335 / CSE-336 Web
  • CSE-358 AI, CSE-381 UI Design
Year 4

Advanced topics

Students reach security, machine learning, NLP, robotics, mobile, microservices, LLMs, and graduation projects.

  • CSE-409 NLP
  • CSE-425 Network Security
  • CSE-445 Machine Learning
  • CSE-476 LLMs, CSE-491 / 492
