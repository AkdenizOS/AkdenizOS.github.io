Why it matters

A modern academic ecosystem,
not just a link list.

AkdenizOS helps students find the right material faster, understand the department roadmap visually, and contribute back without friction.

Organised by real academic progression

Repositories map to the actual student journey — from first-year fundamentals to advanced senior electives and graduation projects.

Built for speed

Jump from course code to repository in seconds — no more hunting through scattered messages and shared drives.

Contribution-first mindset

Anyone can enrich the ecosystem — open an issue, submit a pull request, or help launch a missing course repository.

More than lecture notes

Simulations, systems, visualisers, management apps, and interdisciplinary student builds — all in one place.

Student knowledge, publicly accessible

An open platform so current and future students can build on shared academic momentum — no login required.

Academic roadmap

Follow the four-year learning journey.

Year 1

Foundations

Programming, physics, mathematics, statistics, and the first building blocks of engineering thinking.

  • CSE-101T / CSE-101L
  • CSE-102T / CSE-102L
  • CSE-105, CSE-111, CSE-112
  • MAT-163, MAT-164
Year 2

Core engineering

Data structures, OOP, databases, computer organisation, discrete math, and digital design.

  • CSE-201 Data Structures
  • CSE-203 OOAD
  • CSE-204 Database Systems
  • CSE-206, CSE-211, CSE-221
Year 3

Systems & specialisation

Networking, software engineering, scripting, web, AI, embedded systems, and interface design.

  • CSE-320 Computer Networks
  • CSE-332 Software Engineering
  • CSE-335 / CSE-336 Web Dev
  • CSE-358 AI · CSE-381 UI
Year 4

Advanced topics

Security, machine learning, NLP, robotics, mobile, microservices, LLMs, and graduation projects.

  • CSE-409 NLP
  • CSE-425 Network Security
  • CSE-445 Machine Learning
  • CSE-476 LLMs · CSE-491/492
